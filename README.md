 # Hymn MIDI Repository

This repository contains MIDI files I’ve created for hymns that are not readily available on sites like HymnSite. The goal is to provide practical, usable MIDI files for congregational singing, especially within the context of the United Methodist Hymnal (UMH).

## Structure

- **Main Folder**
  - Contains completed MIDI files.
  - Each file includes:
    - A proper **hymn introduction** (typically based on the final phrase).
    - The correct **number of repeats** to accommodate all verses as they appear in the UMH.
  - These are intended to be ready for real congregational use without modification.

- **WIP Folder**
  - “Work in Progress” files.
  - These MIDIs are **missing an introduction** and may not yet include finalized verse repeats.
  - They are usable for reference but not yet ideal for leading singing.

## Purpose

Many available hymn MIDIs are either incomplete, unavailable, or not structured for real congregational flow. This project aims to:
- Fill gaps where MIDIs don’t exist
- Provide musically practical arrangements
- Preserve a consistent and predictable structure for worship use

## Notes

- MIDIs are arranged with usability in mind, not just playback accuracy.
- Introductions are intentionally simple and clear to support congregational entry.
- Repeats are structured to match typical hymn usage rather than requiring manual looping.
